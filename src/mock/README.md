# Gym HR & Membership System - Mock Data Templates (Stage 1)

This directory is prepared to store local JSON schemas and static arrays when business modules are active.

## Intended Schema Files:
1. `clients.json`: Array of member accounts, tracking registration indices, check-in averages, and billing.
2. `coaches.json`: Roster of accredited trainers, specializations, and operational statuses.
3. `inventory.json`: Equipment, apparel, and beverage items.
4. `payments.json`: Cleared, pending, and failed receipt references.
5. `workouts.json`: Routine templates containing exercise repetitions and sets.
6. `diets.json`: Meal schedules and target macronutrients.
7. `notifications.json`: Alert center history events.
8. `transfers.json`: Roster change logs.
